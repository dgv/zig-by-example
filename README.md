<p align="center">Content and build toolchain for <em><a href="https://zig-by-example.com">Zig by Example</a></em>.</p>

## Building

To build the site:

```sh
tools/build
```

To build the site continuously in a loop:

```sh
tools/build-loop
```

To serve the site locally:

```sh
tools/serve
```
