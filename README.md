## Building

To build the site you'll need Go installed. Run:

```sh
tools/build
```

To build continuously in a loop:

```sh
tools/build-loop
```

To see the site locally:

```sh
tools/serve
```

and open `http://127.0.0.1:8000/` in your browser.
